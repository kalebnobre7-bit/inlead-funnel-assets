# Inlead Funnel Assets

Imagens hospedadas para os funis da plataforma Inlead. Servidos via GitHub Pages para uso direto pela URL no editor.

## Estrutura

```
inlead-funnel-assets/
├── essenza/                  # Funil Essenza 2.0 (id 130274)
│   └── step-01-bem-vindo-hero.png
├── betterfit/                # Funil BetterFit 2.0 (id 129746)
└── README.md
```

## Convenção de nomes

```
step-NN-<step-slug>-<contexto>.png
```

Exemplos:
- `step-01-bem-vindo-hero.png` — imagem hero da step 1
- `step-04-yoga-facial-q.png` — imagem da pergunta da step 4
- `step-22-venda-f-mockup.png` — mockup do produto na venda feminina

## Como usar

URL pattern: `https://<user>.github.io/inlead-funnel-assets/<funnel>/<file>`

No editor da Inlead, no painel direito da imagem, clicar na aba **URL** e colar a URL raw.
